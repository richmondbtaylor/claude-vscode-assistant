# Platform Specifications

## YouTube — Long-Form Video

### Script Structure
- **Hook (0:00–0:03):** One sentence. State the biggest payoff or most surprising claim. Don't tease — deliver.
- **Body:** Segment into 3-5 chapters. Insert a pattern interrupt every 15-20 seconds (question, stat drop, visual shift, or reframe).
- **CTA:** Both verbal ("comment below", "subscribe") and visual (on-screen card/end screen). Place at the 70% mark and again at the end.

### YouTube Description
- **First 200 characters:** Primary keyword + clear value statement (this is what appears in search previews)
- **Length:** 200-500 words
- **Structure:** Hook paragraph → Timestamps → Resource links → About section → Subscribe CTA
- **Timestamps format:** `0:00 Introduction` / `2:15 Why GTM alignment fails` / `5:42 The fix`
- Link every resource mentioned in the video
- Include one engagement prompt (question for comments)

### Thumbnail Concepts
- Text overlay: 3-6 words, bold, high contrast (white on dark or black on bright)
- Visual hook: face with strong expression OR bold graphic — not both cluttered together
- Curiosity gap: headline implies a surprising answer without giving it away
- Dimensions: 1280×720

## YouTube Shorts

### Script Structure
- **0:00–0:03:** Immediate hook — state the payoff before any context
- **Middle:** One focused idea only. No tangents. Tight delivery.
- **End:** Pattern: reveal → reinforce → CTA or loop back to hook
- **On-screen text:** Suggest text overlays for key words/stats — Shorts viewers often watch muted
- **Pacing:** Cut ruthlessly. Every second must earn its place.

### Retention-First Formats
- **PAS:** Name the problem → make it hurt → drop the solution in the last 5 seconds
- **BAB:** Show the before state → flash the after → explain the bridge
- **List:** "3 things that [result]" — number on screen, one point per second

### Specs
- Length: 30-60 seconds
- Aspect ratio: 9:16 (vertical)
- On-screen text suggestions included with every script

## LinkedIn — Single Post

### Format
- **Length:** 150-300 words
- **Line breaks:** Every 1-2 sentences — LinkedIn rewards whitespace, punishes walls of text
- **Hook (line 1):** Must stand alone before "see more" cutoff. Bold statement, surprising stat, or direct question. No setup, no context — just impact.
- **Body:** One idea per paragraph. Lead with the insight, follow with the evidence.
- **CTA:** One specific next step at the end. Not "what do you think?" — something actionable or a direct question that invites a real answer.
- **Hashtags:** 3-5 maximum, placed at end, relevance over volume
- **Tone:** Professional but human. Write like a senior practitioner sharing what they've learned — not like a brand announcement.

### Post During
Business hours EST (Tuesday–Thursday, 8am–10am or 12pm–1pm for highest reach)

### Engagement Hooks (choose one per post)
- Contrarian take: "Most [role]s get this wrong..."
- Specific number: "After reviewing 200 GTM plans, I found..."
- Direct challenge: "If your pipeline is under target, it's probably not a leads problem."
- Before/after: "6 months ago our churn was X. Here's what changed."

## LinkedIn Carousel

### Structure
- **Slide 1:** Hook slide — bold headline (5-8 words), subheadline that promises value
- **Slides 2-8:** One idea per slide. Headline + 2-3 bullet points or a single stat with context.
- **Final slide:** Summary of key points + one CTA (follow, comment, or DM)

### Per-Slide Specs
- Headline: 5-10 words, bold
- Body: 15-30 words max — tight, scannable
- Visual suggestion: icon, chart type, or image description
- Consistent color scheme across all slides

### Flow Logic
Each slide should make the reader want to swipe to the next. Use a "reveal" structure — don't front-load all the value in slide 1.

### Dimensions
1080×1080 (square) or 1080×1350 (portrait) — choose one, stay consistent across the carousel

## Infographic

### Content Hierarchy
1. **Headline:** The single most important takeaway (under 10 words)
2. **Subheadline:** Context or framework name (under 15 words)
3. **Data points:** 3-7 stats or facts with context labels. Format: `[Number] — [What it means]`
4. **Source citations:** Under each stat or in a footnote section
5. **Visual recommendations:** Chart type (bar, pie, timeline, comparison), icon suggestions, color emphasis

### Stat Callout Format
```
[METRIC]
[1-sentence explanation of what this means and why it matters]
Source: [Name, Year, Link if available]
```

### Visual Suggestions
- Use green/positive colors for growth stats
- Use contrast colors to highlight the most important number
- Recommend icon categories (e.g., "dollar sign icon", "arrow up icon", "people icon") — don't leave visual direction vague

## Quote Cards

### Content
- Quote: verbatim, 15-50 words, filler words removed
- Attribution: [Speaker Name], [Title/Role], [Company if relevant]
- Timestamp: include for video sources (for verification)

### Dimensions and Layout
| Platform | Size | Text placement |
|----------|------|----------------|
| Instagram | 1080×1080 | Center or left-aligned, 40-60% card width |
| LinkedIn | 1200×628 | Left-aligned, text in left 60% |
| Stories | 1080×1920 | Centered, upper or middle third |

### Design Direction
- Background: solid brand color or subtle gradient (specify hex if known)
- Quote font: bold sans-serif
- Attribution font: lighter weight, smaller size
- Logo: corner placement, consistent across cards

## Stat Graphics

### Content
- Data point: single metric with number and unit (e.g., "47% decrease in ramp time")
- Context: one sentence explaining significance — not just what the number is, but why it matters
- Source: name + year + link if available

### Design Direction
- Number: largest element on the card — dominant visual hierarchy
- Context text: secondary size, below or beside the number
- Visual metaphor: suggest a simple icon or graphic that represents the metric (e.g., "upward arrow" for growth, "clock" for time savings)
- Dimensions: match quote card dimensions for consistency
